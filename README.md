## HTML Requirements

  ### 1. Home Page ('index.html')
    - Sementic HTML5 structure ('<header>', '<nav>', '<main>', '<section>', '<footer>')
    - Includes company logo, navigation.
    - Grid-based services section with 4 cards
    - About preview section with mission, vision
    - Footer with contact info, social links, and copyright

  ### 2. About Page ('about.html')
    - Consistent header and footer
    - Company history timeline using ordered list
    - Team section with photos, names, positions
    - Values section using sementic markup

  ### 3. Contact Page ('contact.html')
    - Contact details (address, phone, email, office hours)
    - Advanced Contact Form:
    - Personal Information (name, email, phone, company)
    - Inquiry Details (service interest, project type etc.)
    - Terms of service agreement and newsletter checkbox
    - submit and reset buttons


## CSS Requirements
  ### File: 'css/styles.css'
    - External CSS file linked in all pages
    - CSS reset/normalize at the top
    - Responsive grid layout

## Technologies Used
  - HTML5
  - CSS
  - Semantic Web Principles
